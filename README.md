# Mactoy

Native macOS app for creating bootable USB drives. Installs Ventoy or flashes raw disk images, without needing a Linux VM or the Ventoy LiveCD.

**Status:** design phase — see [docs/specs/2026-04-18-mactoy-design.md](docs/specs/2026-04-18-mactoy-design.md).
